# 📘 Breast Cancer Prediction Using Machine Learning (Random Forest)

## 🧠 Introduction
Breast cancer is one of the most common and life-threatening diseases affecting women worldwide. Early detection plays a crucial role in improving survival rates and effective treatment.

This project uses Machine Learning (Random Forest) to predict whether a tumor is **malignant** or **benign** based on medical data.

---

## 🎯 Objective
- Build a predictive model for breast cancer detection  
- Perform Exploratory Data Analysis (EDA)  
- Apply data preprocessing techniques  
- Train using Random Forest algorithm  
- Evaluate model performance  

---

## ❓ Problem Statement
Manual diagnosis can be time-consuming and error-prone when dealing with large datasets. This project provides an automated system to assist in accurate and fast prediction.
